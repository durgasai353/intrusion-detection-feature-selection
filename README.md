# Performance Analysis of Intrusion Detection Systems Using Feature Selection

## Project Overview
This project analyzes the performance of Intrusion Detection Systems (IDS) using machine learning with feature selection techniques. The goal is to improve detection accuracy while reducing computational complexity.

## Technologies Used
- Python
- Machine Learning
- Feature Selection Techniques
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Project Structure
ABSTRACT – Project abstract and summary  
BACKEND – Machine learning model and dataset processing  
FRONTEND – User interface or visualization scripts  
DOCUMENTATION – Complete project report  
PPT – Project presentation slides  

## Features
- Data preprocessing and cleaning
- Feature selection to improve model performance
- Machine learning model training
- Performance evaluation using accuracy metrics

## Dataset
The dataset used for this project contains network traffic data used for intrusion detection.

## Results
The model evaluates IDS performance using metrics like:
- Accuracy
- Precision
- Recall
- F1 Score
